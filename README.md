# Homelab Network Documentation

This repository contains documentation for my personal homelab and network infrastructure.

The purpose of this project is to document every component of my network, including hardware, virtual machines, network services, IP addressing, VLANs, and topology. It serves as a central source of truth for my environment while also documenting configuration decisions, future upgrades, and changes over time.

## Repository Contents

- Physical network topology
- Logical network topology
- Hardware inventory
- Network devices
- Proxmox host documentation
- pfSense firewall configuration
- Virtual machines and containers
- Network services (Pi-hole, Nextcloud, etc.)
- IP addressing and DHCP reservations
- VLAN documentation
- Firewall and NAT rules
- Future improvements and upgrade plans

## Goals

- Maintain accurate documentation of my network.
- Practice industry-standard documentation techniques.
- Track changes as my homelab evolves.
- Demonstrate networking and systems administration knowledge.

> **Note:** Sensitive information such as public IP addresses, passwords, API keys, and serial numbers has been omitted or redacted.
