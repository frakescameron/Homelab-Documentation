# pfSense Firewall

The firewall is the core of my home network. It is a custom-built firewall appliance running pfSense Community Edition. Every device on my network routes through this system, making it responsible for firewalling, routing, NAT, VLANs, DHCP, and DNS forwarding.

I chose this appliance because it offered enterprise-grade networking hardware at a fraction of the cost of a new firewall while still providing enough performance for my current and future homelab projects.

## Specifications

| Specification | Value |
|--------------|-------|
| Firewall OS | pfSense Community Edition |
| CPU | Intel Atom C3558 (4 Cores / 4 Threads @ 2.20 GHz) |
| AES-NI Support | Yes |
| Memory | 8 GB DDR3 |
| Storage | 120 GB SSD |
| Ethernet | 6 × Gigabit Intel NICs |
| NIC Chipsets | 2 × Intel I210, 4 × Intel I354 |
| USB | 2 × USB |
| Console | RJ45 Console Port |
| Form Factor | 1U Rackmount Appliance |
| Purchase Price | ~$95 |

## Responsibilities

This appliance currently performs the following functions:

- Stateful firewall
- IPv4 NAT
- Inter-VLAN routing
- DHCP server
- DNS Resolver / Forwarder
- VLAN segmentation
- Firewall rule management
- Firewall logging and traffic monitoring
- Remote administration

## Planned Features

As my homelab grows, I plan to configure:

- WireGuard VPN
- Intrusion Detection/Prevention (Suricata)
- Traffic shaping (QoS)
- High Availability (HA)
- IPv6

## Why I Chose It

Most consumer routers combine routing, switching, wireless networking, and firewall functionality into a single device. I wanted to separate these functions into dedicated hardware to gain experience with enterprise networking concepts while improving the flexibility of my home network.

This appliance provides six Intel Gigabit interfaces, allowing me to dedicate physical interfaces to WAN, LAN, and future network segments without relying solely on VLAN trunking.

For approximately $95, it offered significantly better value than purchasing a comparable Netgate appliance while still supporting all of the pfSense features I wanted to learn.

## Pros

- Enterprise firewall software
- Six Intel Gigabit network interfaces
- Reliable Intel I210 and I354 NICs
- AES-NI hardware encryption acceleration
- Plenty of storage for logs and packages
- Rackmount design keeps the homelab organized
- Low power consumption compared to a full desktop PC
- Excellent value for the price

## Cons

- No support for 2.5 GbE or faster networking
- Older DDR3 platform
- Intel Atom processor is not ideal for very high VPN throughput
- Refurbished hardware means cosmetic wear is expected
- Community Edition does not include some Netgate Plus features
