# ASUS ROG Rapture GT-AC2900

The ASUS GT-AC2900 serves as the wireless access point for my home network. Operating in Access Point mode, it provides wireless connectivity while leaving all routing, firewall, DHCP, and DNS responsibilities to the pfSense firewall.

Using a dedicated firewall and access point creates a cleaner network design by separating routing and wireless functions into dedicated devices.

## Responsibilities

The access point currently performs the following functions:

- Provides wireless network access
- Broadcasts 2.4 GHz and 5 GHz Wi-Fi networks
- Bridges wireless clients to the wired LAN
- Supports WPA2/WPA3 wireless security
- Extends wireless coverage throughout the home

## Planned Features

As my homelab grows, I plan to configure:

- Dedicated IoT wireless network
- Guest Wi-Fi network
- VLAN-aware SSIDs
- Roaming with additional access points (if needed)

## Why I Chose It

I originally purchased the GT-AC2900 as my primary router. After introducing a dedicated pfSense firewall into my homelab, I converted it to Access Point mode.

This allows the device to focus solely on providing wireless connectivity while pfSense manages routing, firewall rules, DHCP, and other network services. Reusing existing hardware also saved money while still providing strong wireless performance.

## Pros

- Strong Wi-Fi 5 performance
- Reliable dual-band wireless connectivity
- Four Gigabit LAN ports
- Simple Access Point mode
- Good wireless coverage
- Reuses existing hardware instead of requiring a dedicated AP

## Cons

- Limited to Wi-Fi 5 (802.11ac)
- No Wi-Fi 6 or Wi-Fi 7 support
- Limited to Gigabit Ethernet
- End-of-life hardware with limited future firmware updates
