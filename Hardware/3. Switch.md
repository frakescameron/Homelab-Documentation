# Cisco Catalyst WS-C2960CG-8TC-L

The Cisco Catalyst WS-C2960CG-8TC-L is the primary managed switch in my homelab. It connects directly to the pfSense firewall and serves as the central distribution point for all wired network devices.

I chose this switch because it provides enterprise-grade Layer 2 switching while allowing me to gain hands-on experience with Cisco IOS, VLANs, trunking, Spanning Tree Protocol (STP), EtherChannel, and other networking concepts covered by the CCNA.

## Specifications

| Specification | Value |
|--------------|-------|
| Manufacturer | Cisco |

## Responsibilities

The switch currently performs the following functions:

- Layer 2 switching
- VLAN segmentation
- 802.1Q trunking
- MAC address learning
- Spanning Tree Protocol (Rapid PVST+)
- Port security
- Interface monitoring
- Device connectivity throughout the network
- PortFast
- BPDU Guard

## Planned Features

As my homelab grows, I plan to configure:

- EtherChannel (LACP)
- DHCP Snooping
- Dynamic ARP Inspection
- Additional VLANs for lab environments

## Why I Chose It

One of my primary goals is to become a Network Engineer, so learning Cisco IOS was an important factor when selecting a switch.

The Catalyst 2960 series is widely used for CCNA training and enterprise networking labs. It supports nearly every Layer 2 technology required for the CCNA while providing the same command-line interface found on larger Cisco switches.

Although it lacks Layer 3 routing capabilities, those responsibilities are handled by my pfSense firewall, making this switch an excellent fit for my network.

## Pros

- Enterprise-grade Cisco hardware
- Full Cisco IOS command-line interface
- Excellent platform for CCNA practice
- Supports VLANs, trunking, STP, and port security
- Reliable Intel-quality switching hardware
- Quiet and low power consumption
- Rack-friendly compact design

## Cons

- Layer 2 only (no dynamic routing)
- No PoE support
- Limited to Gigabit Ethernet
- Older hardware platform
- Cisco Smart Licensing is not available on newer software versions
